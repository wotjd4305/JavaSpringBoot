#목표
1)지금까지 정적으로 했다면 이제는 동적으로(HTML 웹 페이지 개발)
2)웹페이지 개발은 MVC패턴을 기반, 그 과정을 알아보기로.

#강의순서
1)컨트롤러 추가 머스텟에 인자 전달
2)회원가입 페이지 구현
3)사용자 목록 페이지 구현
4)원격서버에 소스 코드 배포
5)이전 상태로 원복후 반복 구현!!(Branch 사용)

#용어
1)push - Origin으로 전송
2)commit - 로컬에 데이터 패키징하여 저장.
3)branch - 독릭접인 작업을 진행하는 개념.
4)merge - 브랜치와 짝으로 여러작업을 병합하여 하나의 브랜치로.
